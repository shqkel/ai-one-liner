# 개인정보 처리방침 (Privacy Policy)

**AI One-Liner** Chrome 확장 프로그램 (이하 "본 확장")의 개인정보 처리방침입니다.

최종 업데이트: 2026-03-28

---

## 1. 수집하는 정보

본 확장은 다음 데이터를 처리합니다:

| 데이터 | 용도 | 저장 위치 |
|--------|------|-----------|
| 사용자가 선택한 텍스트 | AI 설명 생성을 위해 OpenAI API로 전송 | 저장하지 않음 (일시적 전송만) |
| OpenAI API Key | API 인증 | 사용자 브라우저 `chrome.storage.sync` |
| 트리거 모드 설정 | 사용자 환경설정 | 사용자 브라우저 `chrome.storage.sync` |

## 2. 수집하지 않는 정보

본 확장은 다음 정보를 **수집하지 않습니다**:

- 브라우징 기록 또는 방문 URL
- 개인 식별 정보 (이름, 이메일 등)
- 쿠키 또는 트래킹 데이터
- 사용 통계 또는 분석 데이터

## 3. 데이터 전송

- 사용자가 텍스트를 선택하면, 해당 텍스트만 **OpenAI API** (`api.openai.com`)로 전송됩니다.
- 전송은 사용자의 명시적 동작(텍스트 선택 또는 클릭)에 의해서만 발생합니다.
- 본 확장은 자체 서버를 운영하지 않으며, OpenAI 외의 제3자에게 데이터를 전송하지 않습니다.

## 4. 데이터 저장

- API Key와 설정값은 Chrome의 `chrome.storage.sync`에만 저장됩니다.
- 이는 사용자의 브라우저 내부에 저장되며, 본 확장 개발자가 접근할 수 없습니다.
- 확장 프로그램을 제거하면 모든 저장된 데이터가 삭제됩니다.

## 5. 제3자 서비스

본 확장은 다음 외부 서비스와 연동됩니다:

- **OpenAI API**: 텍스트 설명 생성 ([OpenAI 개인정보 처리방침](https://openai.com/privacy))
- **AI 서비스 바로가기** (선택적): ChatGPT, Claude, Gemini, Grok, Perplexity — 사용자가 버튼을 클릭할 때만 해당 서비스의 새 탭이 열립니다.

## 6. 권한 사용 사유

| 권한 | 사유 |
|------|------|
| `activeTab` | 현재 탭에서 선택한 텍스트를 읽기 위해 |
| `storage` | API Key 및 설정을 브라우저에 저장하기 위해 |
| `contextMenus` | 우클릭 메뉴에서 AI 설명 기능을 제공하기 위해 |
| Content Script (`<all_urls>`) | 모든 웹페이지에서 텍스트 선택 감지를 위해 |
