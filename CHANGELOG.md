# Changelog

모든 주요 변경사항을 이 문서에 기록합니다.

## [1.1.0] - 2026-04-02

### Changed
- 자동 트리거 모드에서 아무 키 입력 시 트리거 취소되도록 개선
- 드래그 중 Ctrl+C 후 mouseup 시에도 트리거가 취소되도록 개선
- 입력 요소(input, textarea, contentEditable)에서 텍스트 선택 시 트리거 방지
- 자동 트리거에 400ms 딜레이 추가하여 복사 동작과의 충돌 방지

## [1.0.0] - 2026-03-28

### Added
- AI One-Liner 크롬 확장 프로그램 최초 출시
- 텍스트 드래그 선택 시 AI 한줄 요약 툴팁 표시
- 자동(auto) / 수동(manual) 트리거 모드 지원
- "더 알아보기" 버튼으로 상세 설명 확장
- 히스토리 네비게이션 (이전/다음 검색 결과)
- 선택 텍스트 하이라이트 표시
- 컨텍스트 메뉴(우클릭) 검색 지원 (PDF 뷰어 대응)
- Ctrl+C 복사 시 자동 트리거 취소 (복사 우선)
- 크롬 웹스토어 배포
